# RentBikes

Es un proyecto maven el cual contiene varias carpetas una de ellas llamada src/main/java que es donde se encuentra el código fuente que contiene un paquete de entidades que tiene 2 clases, una de la factura que genera la renta con los datos del cliente y el precio final de la misma y otra clase con los datos del cliente, otro paquete tiene una clase con la excepción que arroja cuando el descuento familiar no se encuentra entre las 3 y 5 rentas. Un paquete de servicio que tiene una clase abstracta renta y el paquete de implementación de ese servicio que contiene las clases con las distintas tipos de rentas que heredan de la clase padre renta y ya que todas tienen el método calcular precio pero lo realizan de diferente forma utilizamos el concepto de polimorfismo.Tambíen hay otro paquete llamado factory que es el patrón de diseño que implemente para instanciar las distintas rentas en los test y por ultimo un paquete de utilidades que tiene una clase llamda Constants que contiene el nombre de las distintas tipos de rentas. Otra carpeta llamada src/main/resources que contiene un archivo de properties con la información de los precios de las rentas.

Los test se corren con la librería de JUnit y se encuentran en la carpeta src/test/java la cual contiene un paquete de test que posee 4 clases que son las distintas tipos de rentas que hay, lo que se realiza basicamente en los test es instanciar un objeto renta con el tipo de renta que es, fecha de entrega, fecha de devolución y los datos del cliente, después se genera una factura y se verifica en el test si el precio de la factura es el correspondiente y en el test familiar se prueba aparte la excepción que arroja si la cantidad de rentas no se encuentra entre 3 y 5.

El coverage es del 94,3%.
